# Microglia-Poster-MAA-Golden-Section-2026-
A repository that encapsulates our work presented at the MAA Golden Section Spring 2026!

## ABM code folder
This contains the Mesa ABM model that our team leveraged to produce the results of our poster. A button in the UI allows us to snapshot an ABM simulation run, including its data and parameters.

## ABM flowchart folder
Contains three flowcharts for each agent in our model. Each flow chart represents the correponding agent's logic per tick. Overall they represent a high level overview of our code.

## data folder
Contains the three data files and parameter json files that contain information on the genotype dependent simmulations ran to generate our results. We use the UI in our code to snapshot the data and json, rename to the APOE genotype we selected for that run,  and use it later for producing figures.

## figures folder
Contains the figures included on our poster, along with the jupyter notebook which can produce the same grahphs if needed. Keep in mind our original files were renamed after being generated from the python function relating to that figure.

## References file
Contains all relevant sources leveraged in our research in APA format.
